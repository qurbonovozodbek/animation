// Sample GSAP code with multiple animations
const masterTL = gsap.timeline({
  defaults: { 
    duration: 1.2, 
    ease: "power3.out" 
  }
});

// Staggered animations
masterTL.from(".hero-title span", {
  y: 100,
  opacity: 0,
  stagger: 0.1,
  duration: 1.5
});

// ScrollTrigger animation
gsap.to(".feature-card", {
  scrollTrigger: {
    trigger: ".features-section",
    start: "top 70%",
    toggleActions: "play none none none"
  },
  y: 0,
  opacity: 1,
  stagger: 0.1,
  duration: 0.8
});

// Morphing SVG animation
const morphSVG = MorphSVGPlugin.pathDataToBezier("#morph-path");
gsap.to("#morph-svg path", {
  duration: 2,
  bezier: {
    values: morphSVG,
    type: "cubic"
  },
  ease: "sine.inOut",
  repeat: -1,
  yoyo: true
});

Frequently Asked Questions

Find answers to common questions about Animotion Pro

What makes Animotion Pro different from GSAP?

Animotion Pro builds on GSAP's powerful animation engine by adding a user-friendly interface, pre-built animation templates, visual timeline editing, and specialized plugins that simplify complex animations. While GSAP provides the core animation capabilities, Animotion Pro makes these advanced features accessible to developers of all skill levels.

Do I need to know JavaScript to use Animotion Pro?

While JavaScript knowledge is helpful for advanced customization, Animotion Pro includes a visual editor that allows you to create complex animations without writing code. Our library of pre-built animations can be customized through a simple interface, and the code generated can be further tweaked if desired.

Is Animotion Pro compatible with React/Vue/Angular?

Yes! Animotion Pro works seamlessly with all major JavaScript frameworks. We provide framework-specific documentation and examples to help you integrate animations into your React, Vue, or Angular projects. The animations are framework-agnostic and work with any modern web technology.

How does the performance compare to CSS animations?

Animotion Pro animations are highly optimized and often outperform CSS animations, especially for complex sequences. The GSAP engine intelligently uses hardware acceleration and avoids layout thrashing. Our performance monitoring tools help you identify and fix any animation bottlenecks in your projects.

Can I cancel my subscription anytime?

Absolutely. There are no long-term contracts, and you can cancel your subscription at any time. You'll continue to have access to Animotion Pro until the end of your current billing period. We also offer a 30-day money-back guarantee if you're not completely satisfied.
